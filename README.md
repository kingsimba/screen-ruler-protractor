# 屏幕量角器 (Screen Protractor)

一个 WPF 桌面应用，提供透明的屏幕浮层量角器和直尺工具，支持全屏任意位置的角度和长度测量。

## 主要功能

### 量角器模式

- **两根可拖动的测量线**：从中心顶点出发的两条射线
- **实时角度读数**：显示两线间夹角（精确到 0.1°）
- **圆弧标记**：橙色圆弧指示测量角度范围
- **三点控制**：
  - 橙色中心点：移动整个量角器
  - 两个蓝色端点：各自旋转对应的测量线

### 直尺模式

- **刻度尺**：从端点 A 到端点 B 的测量刻度
- **自动 DPI 检测**：根据显示器 EDID 物理尺寸自动转换厘米（cm）显示
- **毫米到厘米刻度**：
  - 每毫米短刻度、每 5mm 中刻度、每厘米长刻度加数字
  - 若无法获取物理 DPI，自动回退像素刻度（每 50px）
- **两端控制**：两个蓝色端点可拖动，直尺本体整体可拖动
- **垂直接近防抖**：用迟滞（双门限）避免直尺接近垂直时刻度方向抖动

## 使用方法

### 启动

```bash
Protractor.exe
```

启动时程序会：

1. 在屏幕中央创建透明浮层
2. 恢复上次保存的模式（量角器/直尺）和工具位置
3. 显示通知区域（右下角）托盘图标

### 交互操作

#### 点击穿透

- 整个屏幕默认**点击穿透**，下方软件照常使用
- 只有光标靠近控制点（或在直尺内部）时，该区域才拦截鼠标，允许拖动
- 下方软件永远不会被量角器和直尺遮挡

#### 量角器模式

| 操作           | 效果                          |
| -------------- | ----------------------------- |
| 拖动橙色中心点 | 整体移动量角器                |
| 拖动蓝色端点   | 旋转对应的测量线              |
| 右键点击橙色点 | 弹出菜单：切换/重置/隐藏/退出 |

#### 直尺模式

| 操作                 | 效果                          |
| -------------------- | ----------------------------- |
| 按住直尺任意位置拖动 | 整体移动直尺                  |
| 拖动任一蓝色端点     | 旋转/缩放直尺                 |
| 右键点击直尺         | 弹出菜单：切换/重置/隐藏/退出 |

#### 托盘菜单

| 菜单项          | 效果                   |
| --------------- | ---------------------- |
| 显示/隐藏量角器 | 隐藏浮窗；再点一次恢复 |
| 切换直尺/量角器 | 在两个工具模式间切换   |
| 重置位置        | 将当前工具移回屏幕中央 |
| 隐藏            | 隐藏工具浮窗           |
| 退出            | 关闭应用               |

双击托盘图标 = 显示/隐藏浮窗

### 快捷菜单

在工具上右键点击，根据当前模式显示菜单：

- **切换为直尺** / **切换为量角器**（取决于当前模式）
- **重置位置** - 将工具回到屏幕中央
- **隐藏** - 隐藏工具浮窗
- **退出** - 关闭应用

## 技术细节

### 架构

- **框架**：WPF (Windows Presentation Foundation)
- **目标框架**：.NET 6.0-windows
- **UI 层**：XAML，支持透明背景 + 点击穿透

### 状态管理

- **独立状态**：量角器和直尺各自维护独立的坐标（`_pVertex/_pEnd1/_pEnd2` vs `_rEnd1/_rEnd2`）
- **持久化**：
  - 保存位置：`%AppData%\ScreenProtractor\state.json`
  - 内容：当前模式、量角器三点、直尺两点
  - 自动保存触发：拖动结束、模式切换、重置位置
  - 容错：读写失败自动回退默认值

### 点击穿透实现

- Win32 扩展窗口样式 `WS_EX_TRANSPARENT`（鼠标事件穿透到下层窗口）
- 定时器（30ms 轮询）检测光标位置：
  - 若靠近控制点或在直尺内部 → 临时关闭穿透，允许拖动
  - 拖动中 → 保持交互式
  - 其他位置 → 重新开启穿透

### 物理测量

- **DPI 获取**：
  - `GetDpiForMonitor` 查询显示器两个 DPI 值：
    - `MDT_EFFECTIVE_DPI`：系统缩放比例（96 DPI 为基准）
    - `MDT_RAW_DPI`：显示器 EDID 上报的物理像素密度
  - 换算公式：`DIP × (有效DPI/96) = 物理像素 ÷ 原始DPI = 英寸 × 2.54 = 厘米`

### 竖直防抖（直尺模式）

- 用迟滞状态 `_rulerFlip` 记住刻度朝向
- 法向量 Y 分量双门限：
  - `Y > +0.20` → 刻度朝下（不翻）
  - `Y < -0.20` → 翻转
  - `|Y| ≤ 0.20` → 保持上次方向
- 避免接近垂直时法向频繁翻转导致视觉抖动

## 编译与运行

### 前置条件

- Visual Studio 2022（或 .NET 6.0 SDK）
- Windows 10 / 11

### 编译

```bash
dotnet build Protractor.csproj -c Debug
```

### 运行

```bash
bin/Debug/net6.0-windows/Protractor.exe
```
